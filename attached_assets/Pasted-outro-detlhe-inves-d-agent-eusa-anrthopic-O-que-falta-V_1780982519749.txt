Sim, a sua intuição está correta! O Quant Brain é alimentado pelos dados de candle da BingX. Ele não depende de TradingView ou API externa paga.

📊 De onde vêm os dados?
O sistema busca os dados diretamente da API pública da BingX.

Fonte Oficial: O backend do seu robô faz requisições para o endpoint da BingX: GET /openApi/swap/v3/quote/klines .

Timeframes: Ele puxa os dados para construir os candles, exatamente como você sugeriu (5m, 15m, etc.) .

O que é Calculado: A partir desses preços (Open, High, Low, Close) e volume, o CandleEdge calcula:

Médias Móveis (EMA 9/21)

RSI (Índice de Força Relativa)

ATR (Volatilidade)

Suporte/Resistência (Ichimoku)

Divergências

🧠 Como funciona o aprendizado?
O Quant Brain aprende de duas formas simultâneas:

Aprendizado Estático (Gráfico): Ele analisa os dados de candle (como os indicadores acima) para gerar um "Score de Mercado" instantâneo. Ele olha para o gráfico e diz: "Com base no RSI e no rompimento da EMA, isso parece uma boa entrada LONG".

Aprendizado Dinâmico (Resultados): Ele analisa os resultados dos seus trades (se você ganhou ou perdeu dinheiro). Se uma configuração deu certo, o sistema aprende que aquela formação (ex: "BTC bullish + RSI < 30") é boa e aumenta o "peso" dela para o futuro.
